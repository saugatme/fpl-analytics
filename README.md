# FPL Analytics

End-to-end data science project using the Fantasy Premier League (FPL) API to analyze player performance and support data-driven FPL decisions.

## Objectives
- Collect and store FPL data via public API
- Perform exploratory data analysis and feature engineering
- Build predictive models for player performance
- Present insights via an interactive Streamlit dashboard

## Tech Stack
Python, Pandas, NumPy, Streamlit, Scikit-learn
